### Moving Average
```sql
USE salesorderssample;

SELECT OrderDate
      ,SUM(OrderTotal) AS OrderDateTotal 
      ,AVG(SUM(OrderTotal)) OVER(ORDER BY OrderDate ROWS BETWEEN 7 PRECEDING AND CURRENT ROW) AS OrderDate7Total
FROM ORDERS
GROUP BY OrderDate
ORDER BY OrderDate
LIMIT 20
```
